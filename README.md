## Project Overview

A supervised ML pipeline for credit card fraud detection using the Kaggle Credit Card
Fraud Detection dataset. The project studies class-imbalance handling (class weighting,
SMOTE, random under-sampling), dynamic decision threshold tuning, and cost-sensitive
evaluation under asymmetric FP/FN business costs. As a novel extension, Isolation Forest
is deployed as an unsupervised out-of-distribution (OOD) detector trained solely on
legitimate transactions to flag anomalous — potentially novel — fraud patterns.

**Problem type:** Binary classification (fraud = 1, legitimate = 0)  
**Dataset:** 284,807 transactions, 30 features (V1–V28 PCA + Time + Amount), 0.17% fraud prevalence

## Results Summary

Four supervised models (LR, DT, RF, GBT), three imbalance strategies, and Isolation
Forest OOD detection evaluated and compared.

## Setup

**Requirements:** Python 3.9 or higher

**1. Clone the repository**

```bash
git clone https://github.com/OmPrajapati2110/EE559-Fraud-Detection.git
cd EE559-Fraud-Detection
```

**2. Extract the dataset**

```bash
unzip creditcard.csv.zip -d data/
```

**3. Create virtual environment and install dependencies**

```bash
python3 -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

**4. Launch Jupyter and run the final notebook**

```bash
jupyter notebook
# Open notebooks/fraud_detection_final.ipynb
# Kernel > Restart & Run All
```

Alternatively, execute headlessly:

```bash
jupyter nbconvert --to notebook --execute --inplace notebooks/fraud_detection_final.ipynb
```
